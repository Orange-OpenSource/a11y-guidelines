Resources

WCAG

For Orange internal accessibility requirements, we have chosen to rely on the Web Content Accessibility Guidelines v2.1 (WCAG 2.1). Those guidelines are international guidelines published by the World Wide Web Consortium (W3C) Web Accessibility Initiative (WAI) task force, which comply with the ISO standard 40500:2012.

All HTML pages made by the Orange group must comply with the AA level criteria of Web Content Accessibility Guidelines (WCAG) 2.1, without blocking point for the user after testing the main features of the site or application.

For technical solutions, you should consult WCAG criteria by project phase and examples sections. You can also refer to an external resource available at Accede-Web references.

RGAA

The RGAA is the French government’s General Accessibility Reference for improving accessibility (in French). It is meant to provide a way to check compliance against WCAG 2.1. If your are familiar with this reference, you can use it to test your site.

An English translation is available for informational purposes only.
