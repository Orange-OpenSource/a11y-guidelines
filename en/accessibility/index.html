Methods and Tools to evaluate accessibility

  • Manual testing: va11ydette by Orange (French), Color Contrast analyser, source analysis
  • Functional testing: NVDA V 2020.3, keyboard navigation

Introduction

This page defines the level of accessibility of the site.
This is a statement of compliance with standard Web Content Accessibility Guidelines (WCAG).
These guidelines define how to make Web content more accessible to people with disabilities.
This statement is part of the Orange commitment towards accessibility as described in the pluri-annual accessibility framework (French).
The assessment has been performed by Orange.

Scope of the assessment

The evaluation has been conducting on the following sample of web pages:

  1. Welcome page:https://a11y-guidelines.orange.com/en/
  2. Accessibility regulations:https://a11y-guidelines.orange.com/en/planning/accessibility-regulations/
  3. Web Test:https://a11y-guidelines.orange.com/en/web/test/
  4. Mobile IOS Test:https://a11y-guidelines.orange.com/en/mobile/ios/test/
  5. Articles:https://a11y-guidelines.orange.com/en/articles/
  6. Articles The Accessible Name in HTML:https://a11y-guidelines.orange.com/en/articles/accessible-name/
  7. Editorial Content:https://a11y-guidelines.orange.com/en/editorial-content/

Conformance status

The audit reports a mean compliance of 97% with WCAG with 1 non-compliance.

A compliance score is measured for each audited page: the score equals the sum of compliant criteria divided by the number of criteria applicable to the page.
