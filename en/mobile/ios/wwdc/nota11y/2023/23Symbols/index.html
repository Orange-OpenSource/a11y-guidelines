WWDC 2023: Create and animate symbols

Find hereafter a detailed summary of two videos that belong to a taxonomy of some WWDC footages.

The original videos are available on the Apple official website (session 10257, session 10258).

Overview #

Since their introduction, the SF Symbols are meant to be consistent, customizable and expressive to provide the best user experience.

As a brand new feature in 2023, animation is now to be considered as a whole part of symbols.

A library of iconography has been made for creating custom symbols with shared design characteristics and accessibility features: the SF Symbols app.

SF Symbols app #

The new SF Symbols 5 app introduces an animation inspector in the sidebar to preview the new animation effects.

The animation effect selection comes in with controls for configuring the animation and with a label under the preview area (“Automatic” or “Modified”) to indicate activated custom settings.
Moreover, the effect name can be copied in code with the appropriate syntax through the button next to the label.

Note that the controls may vary according to the selected animation effect.

Custom symbols #

Both for the basic variable colors and the pulse animations, a new control is available in the annotation list to specify which layers should be impacted when the By Layer setting is ticked.

Based on the motion with the bounce animation, another new feature in annotation is introduced for grouping layers that animate their sublayers altogether.

It's important to keep in mind that the motion animations are only for SF Symbols 5.

Symbol effects #


As a feature of the Symbols framework, the dot-separated name writing renders the effects with specific options for some of them.

Each of these effects has its own protocol to describe four distinct behaviors highlighted in SwiftUI through a new view modifier.

  • Indefinite
  • Transition
  • Content Transition

Enhancements #

With SwiftUI, the symbol effects may be disabled throughout the view hierarchy via the symbolEffect and the symbolEffectsRemoved modifiers.

Launching the animation when needed is based on different approaches according to the environments.

The use of variable values has become easier for SwiftUI, AppKit and UIKit.

UIBarButtonItem and some UIKit controls take advantage from the symbol effect like the UISlider.
