Tooltip

Introduction

Tooltips are part of the components conventionally found on websites. HTML provides the ability to display tooltips via the title attribute. That being said, this solution is unfortunately not very attractive since it is impossible to change its appearance. The most common solution is to create tooltips from standard HTML elements (span, div…) whose appearance can be adapted to your needs using CSS. However, for these tooltips to be accessible for users who do not use a mouse, some precautions must be taken.

The easiest solution

The easiest way to display tooltips is to use a Javascript component already accessible. This is the case of the tooltip component available in the Boosted library .

Example

Snippet example


            <form>
                  <div class="form-group">
                        <label for="serial-number">
                              Please enter the serial number
                              <span tabindex="0" class="icon-info" data-toggle="tooltip" data-placement="right" title="on the back of the device"></span>
                        </label>
                        <input type="email" class="form-control" id="serial-number">
                  </div>                              
                  <button type="submit" class="btn btn-default">Envoyer</button>
            </form>

Note: the tabindex attribute is required when the tooltip is used on a non-focusable element (span). Without this attribute, the tooltip would not be accessible from the keyboard.
