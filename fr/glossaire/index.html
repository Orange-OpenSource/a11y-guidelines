Glossaire

AIDES TECHNIQUES
Une aide technique est un matériel ou logiciel utilisé par une personne en situation de handicap destiné à compenser, limiter ou neutraliser la déficience.
TEXTE DE GRANDE TAILLE
Le texte de grande taille est défini comme du texte de 24px ou plus, ou du texte en gras de plus de 19px
API
Application Programming Interface, interface de programmation
ARIA
Accessible Rich Internet Applications, applications internet riches accessibles
AT
Assistive Technology, technologie d’assistance
CSS
Cascading Stylesheets, feuilles de style en cascade
DINSIC
Direction Interministérielle du Numérique et du Système d’Information et de Communication
DOM
Document Object Model, modèle d’objet document
EASE
e-Accessibility Solutions for Everyone, service d’accessibilité numérique chez Orange
EULA
End-User Licence Agreement, contrat utilisateur final
HTML
HyperText Markup Language, langage de marquage hypertexte
HTML5
HyperText Markup Language version 5, langage de marquage hypertexte version 5
ID
Identifier, identifiant
IDE
Integrated Development Environment, environnement de développement intégré
IE
Microsoft Internet Explorer
iOS
Apple’s Operating System for mobile and tablet, système d’exploitation d’Apple pour mobile et tablette
IP
Internet Protocol, protocole internet
IPA
International Phonetic Alphabet
ISO
International Organization for Standardization, organisation internationale pour la standardisation
JAWS
Job Access With Speech, lecteur d’écran
JO
Journal Officiel
JS
Javascript
KOTLIN
Kotlin langage, Kotlin est le second langage utilisé pour programmer sur les mobiles Android
MIT
Massachussetts Institute of Technology, institut de technologie du Massachussetts
NVDA
NonVisual Desktop Access, lecteur d’écran
OSX
Apple’s Operating System version 10, système d’exploitation d’Apple version 10
PC
Personal Computer, ordinateur personnel
PDF
Portable Document Format, format de documents portables
RGAA
Référentiel général d’Amélioration de l'Accessibilité
SDK
Software Development Kit, kit de développement logiciel
SEO
Search Engine Optimisation, optimisation pour les moteurs de recherche
SVG
Scalable Vector Graphics, graphique vectoriel adaptable
TBD
To Be Done, à faire
TDD
Test Driven Development, tests unitaires
TICS
Technologies de l'Information et de la Communication
TTS
Text To Speech, conversion texte vers parole
UI
User Interface, interface utilisateur
URL
Universal Resource Locator, emplacement d’une ressource
W3C
World Wide Web Consortium
WAI
Web Accesibility Initiative, initiative pour l’accessibilité web
WAI-ARIA
Web Accesibility Initiative: Accessible Rich Internet Applications, applications internet riches accessibles
WCAG
Web Content Accessibility Guidelines, recommandations d’accessibilité des contenus web
XHTML
eXtensible HyperText Markup Language, langage de marquage hypertexte extensible
XML
eXtensible Markup Language, langage de marquage extensible
XPath
XML Path Language, langage de parcours XML
