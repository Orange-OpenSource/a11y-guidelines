Les info-bulles (tooltips)

Préambule

Les info-bulles font partie des composants que l’on retrouve classiquement sur les sites web. Le langage HTML a prévu la possibilité d’afficher des info-bulles via l’attribut title. Cela dit, cette solution n’est malheureusement pas très esthétique puisqu’il est impossible de modifier son apparence. La solution consiste donc bien souvent à créer des info-bulles à partir d’éléments HTML classiques (span, div…) dont l’apparence pourra être adaptée aux besoins à l’aide de CSS. Cependant pour que ces info-bulles soient accessibles pour les utilisateurs qui n’utilisent pas de souris, certaines précautions doivent être prises.

La solution simple

La solution la plus simple pour afficher des info-bulles consiste à utiliser un composant Javascript déjà accessible. C’est le cas des info-bulles de la librairie Boosted.

Exemple

Code de l’exemple


      <form>
            <div class="form-group">
                  <label for="numero-serie">
                        Veuillez saisir le numéro de série
                        <span tabindex="0" class="icon-info" data-toggle="tooltip" data-placement="right" title="inscrit au dos de l’appareil"></span>
                  </label>
                  <input type="email" class="form-control" id="numero-serie">
            </div>
            <button type="submit" class="btn btn-secondary">Envoyer</button>
      </form>

Place aux tests

Déplacer le focus sur l’icône « i » à l’aide de la touche Tab.
L’info-bulle apparaît.
Naviguer dans l’exemple à l’aide d’un lecteur d’écran.
Le texte de l’info-bulle est correctement vocalisé.

À noter : l’attribut tabindex est nécessaire lorsque l’info-bulle est utilisée comme ici sur un élément non focusable (span). Sans cet attribut, l’info-bulle ne serait pas accessible au clavier.
